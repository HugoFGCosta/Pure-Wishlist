# Suggested Commands

## Development
- `shopify app dev` — Start dev server (tunneled, with HMR)
- `npm run build` — Build for production (`react-router build`)
- `npm run start` — Start production server

## Shopify CLI
- `shopify app deploy` — Deploy app + extensions to Shopify
- `shopify app env show` — Show env vars
- `shopify app generate extension` — Scaffold new extension

## Code Quality
- `npm run lint` — ESLint with cache

## Database
- Migrations in `supabase/migrations/` — run via Supabase dashboard

## System Utils
- `git`, `ls`, `grep`, `find` — standard Linux tools
